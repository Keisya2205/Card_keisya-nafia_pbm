# 💖 Cute Card App

A Flutter application for creating cute profile cards with photos, names, and quotes.

## ✨ Features
- Upload photo
- Input name
- Input quote
- Save cute card
- Gallery page
- Supabase integration

## 🛠️ Built With
- Flutter
- Dart
- Supabase
